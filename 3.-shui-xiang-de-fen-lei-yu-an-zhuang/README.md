# 3. 水箱的分类与安装

### 一、承压保温水箱的分类

水箱按照用途可分类为：

* 缓冲水箱
  * 单循环缓冲水箱
  * 双循环缓冲水箱
* 热水水箱
  * 无盘管热水水箱
  * 有盘管热水水箱
    * 单盘管
    * 双盘管
    * 多盘管
    * 氟循环水箱

关于盘管水箱详见 [3.3 热水水箱\(带盘管\)](3.3-re-shui-shui-xiang-dai-pan-guan.md) ，氟循环水箱不在以下讨论之列，详见 [3.4 氟循环水箱](3.4-fu-xun-huan-shui-xiang.md)。
